# Parking Lot System

A console-based Parking Lot Management System developed in Java using Object-Oriented Programming (OOP) principles. The application simulates real-world parking lot operations, including vehicle parking, spot allocation, and parking space management.

## Features

- Park vehicles in available parking spots
- Manage different types of vehicles
- Allocate parking spots based on vehicle type
- Track parking spot availability
- Display parking lot status
- Handle full parking lot scenarios

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## Project Structure

```
parking-lot/
│
├── src/
│   ├── ParkingLot.java        # Core parking lot operations
│   ├── ParkingSpot.java       # Represents individual parking spots
│   ├── ParkingSpotType.java   # Defines parking spot categories
│   ├── Vehicle.java           # Vehicle details and operations
│   ├── VehicleType.java       # Defines vehicle categories
│   ├── App.java               # Main entry point
```

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Abstraction
- Composition
- Modular Design

## Functionalities

- Add vehicles to the parking lot.
- Assign parking spots based on vehicle type.
- Check parking spot availability.
- Display the current status of the parking lot.
- Remove vehicles and free occupied spots.
- Handle scenarios when no parking spots are available.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/bhavijaiswal24/parking-lot.git
```

2. Navigate to the project directory:

```bash
cd parking-lot
```

3. Compile the Java files:

```bash
javac src/*.java
```

4. Run the application:

```bash
java App
```
